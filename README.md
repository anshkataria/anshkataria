### About Me

Penultimate Software Engineering student at UQ who genuinely loves problem solving and building scalable systems. I work across the full stack — from backend APIs and distributed systems to frontend interfaces — and I'm currently deep in the **quant and trading** space, building out projects like a **Limit Order Book** in C++ and financial risk models in Python.

I'm passionate about **backend development** and the Spring ecosystem — Spring Boot, Microservices, Data JPA, Hibernate — and I care a lot about writing clean, efficient code. A big believer in continuous learning, I spend time reading books on Spring architecture, C++ for quant, and finance to sharpen my craft.

I'm a huge fan of the **command line**, Linux, and Unix systems programming. University has also shaped how I work in teams — Agile sprints, structured Git workflows, and code reviews are second nature to me. Alongside all of this, I'm exploring **machine learning and generative AI**, curious about where intelligent systems meet real infrastructure.

Let's go.
